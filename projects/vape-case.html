Vape Lock Case

Personal Project · Mechanical + Hardware · Work in Progress

This project explores a combined mechanical and hardware-based system designed to reduce vaping through physical constraint and behavioral feedback rather than pure restriction. The system is intended to regulate access and usage over time instead of relying on complete denial.

The core idea is to place the vape inside a controlled enclosure that interfaces with a mobile device, allowing usage to be gradually reduced in a manner similar to phone screen-time controls.

System Concept

  • Mechanical enclosure to physically contain the vape device
  • Embedded hardware to interface with the vape's power or control circuitry
  • Phone-connected logic to regulate usage over time
  • Ability to reduce output intensity or fully disable the device

Current Focus

  • Mechanical design and tolerancing for the enclosure
  • Exploring non-destructive hardware interfaces
  • Designing safe and reversible control mechanisms
  • Defining user interaction around gradual reduction
